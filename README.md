# 🚀 Proyecto Destacado

## 📊 Análisis del Mercado de Vehículos Usados

Proyecto de análisis exploratorio de datos enfocado en entender los factores que influyen en el precio de vehículos usados.

En este proyecto analicé variables como:

• kilometraje  
• antigüedad del vehículo  
• tipo de combustible  
• condición del vehículo  
• tiempo que el vehículo permanece listado  

A través del análisis exploratorio de datos se identificaron patrones importantes que ayudan a comprender cómo diferentes características afectan el precio en el mercado.
